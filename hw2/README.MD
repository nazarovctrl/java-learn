This is a Java project

> **Note**<br />
> You need to have these dependencies to be installed:<br />
>
> - Java 17
> - Maven


## Instructions for running the app
### 1. Creating .jar 

```bash
mvn package
```

### 2. Run
Open target folder
```bash
cd target
```
run .jar file
```bash
java -jar hw2-1-jar-with-dependencies.jar
```
